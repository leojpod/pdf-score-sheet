PDF score sheet generator for hockey games

A couple of years back, I spent quite some time behind the board of a hockey field (that is when I wasn't on it myself). What I remember from that period is how freezing I was and how complicated it was to fill up the game sheet, to enter correctly the name of all the players (without spell mistakes) and how cold where my fingers when I had to write down every goal or penalty for all these hockey games. At the end of every game I would swear “why isn’t there a software to do that? I could type with gloves and at least I would not need to start over if I make a mistake!” Well it is now thing done!

The small web application whose code is freely accessible here and can be seen working live here handle this particular task. No ninja coding required for that. A simple jquery mobile interface handle the input submission and a single HttpServlet receive them in json format, parse them and feed them to the pdf generator. The BFO pdf library make it easy to use a base pdf and to draw text wherever needed to fill up the gap! Happy coding!
